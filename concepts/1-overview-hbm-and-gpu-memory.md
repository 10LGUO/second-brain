# HBM and GPU Memory Hierarchy

GPU memory is organized in a hierarchy from slow, high-capacity off-chip memory down to fast, small on-chip caches and registers immediately adjacent to compute units. Understanding this hierarchy is essential for memory optimization and operator development in AI infra.
